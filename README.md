## Daedalus
### Jonkershoek Nature Reserve Project_713
This project develops a wildfire prediction model and Fire Danger Index (FDI) for Jonkershoek Nature Reserve using historical fire records, weather data, and topographic variables.

## Methodology
* Applied Random Forest to identify the most influential variables driving wildfire occurrence
* Implemented a Support Vector Machine (SVM) model for classification
* Generated spatial outputs to visualize fire risk distribution
* Evaluated model performance using classification metrics and comparative analysis
## Key Outputs
* Fire Danger Index (FDI) map
* Wildfire risk prediction maps
* Model performance reports (classification metrics)
* Comparative analysis of fire likelihood across the study area
## Tech
*rasterio, geopandas, shapely, pandas, numpy, scikit-learn, matplotlib, seaborn, datetime*
